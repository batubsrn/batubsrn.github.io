Pokemon Type Counters

What & Why?

Siplified interface for Pokemon Type match-ups just type and found out which types are stronger against which.

Why? Well because as loyal pokemonGo players we got tired of type charts trying to see rows and columns.

There are 18 types in pokemon, with quick math we need to be able to memorize almost 300 different effectiveness which could be challenging to keep track of.


Which is where our tool will come in handy, helphing you to easily learn what pokemon is weaker against which type.
